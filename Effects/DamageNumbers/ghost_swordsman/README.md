# Ghost Swordsman Damage Numbers

Generated combat damage digits for the ghost_swordsman visual style.

- Canvas: 64x64 transparent PNG per digit.
- Variants: normal, critical.
- Style notes: DNF-like Korean fantasy, dark ink outline, crimson ghost-hand rim, amber/gold normal damage, hot red/white critical damage.
- Palette: only Resurrect-64 RGB values are used for non-transparent pixels.
- Source: deterministic local generator, derived from the in-project ghost_swordsman frame style.

Files:
- normal/digit_0.png ... normal/digit_9.png
- critical/digit_0.png ... critical/digit_9.png
- manifest.json
